麥肯錫思考框架

探索專業顧問的思維方式

7S 框架

分析組織的七個關鍵要素：

  • Strategy (策略)
  • Structure (結構)
  • Systems (系統)
  • Shared Values (共同價值觀)
  • Style (風格)
  • Staff (員工)
  • Skills (技能)
→

MECE 原則

Mutually Exclusive, Collectively Exhaustive

相互獨立，完全窮盡的分析方法，確保：

  • 沒有重疊（相互獨立）
  • 沒有遺漏（完全窮盡）
→

問題金字塔

自上而下的思考結構：

  • 核心論點
  • 關鍵論據
  • 支持數據
→

3C 分析

市場競爭力分析框架：

  • Company (公司)
  • Competitors (競爭者)
  • Customers (顧客)
→

BCG矩陣

產品組合分析工具：

  • 明星產品
  • 金牛產品
  • 問題產品
  • 落水狗產品
→

SCP範式

產業組織分析框架：

  • Structure (結構)
  • Conduct (行為)
  • Performance (績效)
→

五力分析

產業競爭力分析模型：

  • 供應商議價能力
  • 購買者議價能力
  • 潛在進入者威脅
  • 替代品威脅
  • 同業競爭程度
→

價值鏈分析

企業價值創造活動分析：

  • 主要活動
  • 支援活動
